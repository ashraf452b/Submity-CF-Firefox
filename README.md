# Submitify CF (Firefox Version)

This extension restores the textarea-based submission feature on Codeforces.

## Installation (Firefox)

1. Download this repository as a ZIP file.
2. Extract the ZIP file.
3. Open **Firefox**.
4. In the address bar, go to:
   `about:debugging`
5. Click **This Firefox**.
6. Click **Load Temporary Add-on**.
7. Select the **manifest.json** file from the extracted folder.

The extension will now be active on Codeforces.
